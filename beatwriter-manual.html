Instruction Manual

beatwriter is a tool that allows you to create rhythm and beats using a grid-based interface. You can write and arrange syllables, and even play your compositions. This manual provides a step-by-step guide on how to use the app effectively.

Table of Contents

  • Getting Started
  • Writing Mode
  • Arranging Mode
  • Playing Mode
  • BPM Slider
  • Toggle Mode

1. Getting Started

Open the index.html file in your web browser. The Beat Writer App interface will be displayed.

2. Writing Mode

In this mode, you can input syllables into the grid cells. Click on a grid cell to begin writing. Press the Spacebar to move to the next cell within the same row. Press the Enter key to move to the first cell of the next row. If a word is entered, it will be split into syllables automatically.

3. Arranging Mode

In this mode, you can arrange syllables within the grid. Click on a cell to start the arrangement process. Click on another cell to move the syllable to that cell. Blanks can be inserted by clicking on empty cells. You can only arrange syllables within the same row.

4. Playing Mode

In this mode, you can play your composed beats. Click the Play button to start playback. The metronome will keep the beat while your syllables play.

5. BPM Slider

The BPM slider adjusts the beats per minute (BPM) for playback. Move the slider to set the desired BPM value. The selected BPM value is displayed next to the slider.

6. Toggle Mode

Click the "Toggle Mode" button to switch between writing and arranging modes. The current mode is displayed at the top of the interface.

Conclusion

The Beat Writer App provides a user-friendly way to write, arrange, and play beats using a grid-based interface. Experiment with different syllables and rhythms to create your own unique compositions. Have fun exploring the app and unleash your creativity!
